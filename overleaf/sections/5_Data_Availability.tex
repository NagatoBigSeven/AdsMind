All benchmark inputs, frozen experimental configurations, runner scripts, and curated summary outputs from this study are publicly available in the AdsMind repository at \url{https://github.com/AI4QC/AdsMind}, with a permanent archive deposited on Zenodo [DOI to be inserted upon publication]. The repository provides the CMU20 and OCD-GMAE62 benchmark manifests, comprehensive summary tables for the full ablation matrices (four LLM backends across five variants), and performance comparisons against Heuristic and Adsorb-Agent baselines. Additionally, we provide the $N=3$ repeated-run stability audit data for OCD-GMAE62. All LaTeX tables in the main text and Supporting Information are generated deterministically from these summary files. While frozen configurations, including model identifiers, MACE-MP-0 parameters, and runtime switches, enable reproduction of these summaries, users should note that hosted LLM services are subject to unannounced updates, which may impact exact numerical parity. Due to their substantial size, raw per-run payloads, relaxation trajectories, and generated structure files are hosted exclusively on Zenodo. Access to LLM backends requires API keys from the respective providers, while MACE-MP-0 checkpoints remain publicly accessible~\cite{batatia2023foundation}.
