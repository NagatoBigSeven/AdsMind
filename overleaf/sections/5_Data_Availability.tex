All agent-side benchmark inputs, frozen configurations, runner scripts, and curated structured outputs used in this study are available in the AdsMind GitHub repository at \url{https://github.com/AI4QC/AdsMind}. The repository contains paper-facing summary tables, backend-comparison artifacts, and LaTeX table exports used in the manuscript under \texttt{research/agent\_eval/} and \texttt{research/results/}, as well as frozen experiment configurations and runner scripts for reproducing the reported benchmark and ablation summaries. Per-run logs, trajectories, and structure artifacts are treated as regenerable execution products. The reported benchmark and ablation summaries can be reproduced from the benchmark manifests, frozen configurations, and runner scripts.

The CMU20 and OCD-GMAE62 benchmark manifests are included in the GitHub repository under \texttt{datasets/}. The CMU20 dataset is based on the Adsorb-Agent benchmark~\cite{ock2026adsorbagent}, while OCD-GMAE62 is introduced in this work as a 62-case benchmark. The 12-case OCD-GMAE62 overlap manifest is provided only for repeated-run reproducibility analysis.

Access to the four LLM backends requires corresponding API credentials for GPT-5.4, Claude~Sonnet~4.6, Gemini~2.5~Pro, and Grok-4. API transport and credential configuration are execution details for reruns rather than benchmark identity dimensions. The MACE-MP-0 calculator checkpoints are publicly available through the \texttt{mace.calculators.mace\_mp} interface~\cite{batatia2023foundation}.
