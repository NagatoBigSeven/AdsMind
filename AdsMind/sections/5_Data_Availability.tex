All agent-side benchmark inputs, frozen configurations, runner scripts, and structured outputs used in this study are stored in the repository under \texttt{research/agent\_eval/} and \texttt{research/results/}. Each benchmark or ablation case directory contains the resolved configuration, a machine-readable \texttt{result.json}, the agent log, and the generated structure artifacts needed to trace the full decision path. The representative ablation summaries, backend-comparison artifacts, and paper-facing tables reported in this manuscript are therefore reproducible directly from the archived repository state.
