# Mechanism Ablation Effects

Advanced summary tables derived from the matched CMU20 and OCD62 basic
experiment matrices.

- `variant_delta_statistics.csv`: paired variant-minus-Full deltas with Wilcoxon tests.
- `variant_delta_statistics.json`: per-pair detail behind the statistics table.
- `cross_backend_agreement.csv`: 4-backend energy spread by dataset, case, and variant.
- `reach_full_within_0.01eV.csv`: whether each non-Full variant reaches an energy no more than 0.01 eV higher than Full.

Regenerate with repository analysis scripts after refreshing the basic experiment summaries.
