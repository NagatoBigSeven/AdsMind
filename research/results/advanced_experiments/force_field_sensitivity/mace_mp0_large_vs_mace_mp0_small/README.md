# MACE-MP-0 Large vs Small

This experiment reruns AdsMind Full with GPT-5.4 using MACE-MP-0 large
(`cuda`, `float64`, dispersion enabled) and compares it with the primary
MACE-MP-0 small benchmark protocol.

Current committed results:

- `cmu20/openai_gpt54_mace_mp0_large/full/`: 20/20 CMU20 cases completed.

There is no paper-facing OCD62 MACE-large subset in this directory.
