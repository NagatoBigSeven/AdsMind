# Paper Artifacts

This directory documents the external paper artifact bundle. It should not
store heavy generated outputs in the Git repository.

For each public release, stage the reproducibility bundle outside the repository
and upload it as:

```text
adsmind-paper-artifacts-v0.1.0.zip
```

Attach the zip to the matching GitHub release. Before manuscript submission,
upload the same zip to Zenodo and cite the resulting DOI in the data
availability statement.

The intended bundle structure is defined in `MANIFEST.md`.
